# Game Reference Folder

For now, this folder only contains the standard set of games. It serves as an easy reference to them, although these same game definitions are also in the `ludax` package, under the [`ludax/games/`](../src/ludax/games/) submodule.

In the future, we plan to also add games to this folder that we don't want to include in the pip package, such as AI-generated games or unit tests for the Ludax package.
